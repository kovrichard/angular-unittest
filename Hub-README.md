# What is this image

This image can be used for Angular development and testing locally and in a CI.

Contents:
- Node.js version: 16
  + Yarn is also available
- Alpine version: 3.11
- @angular/cli version: 8.1.0

# How to use this image

```
FROM richardkovacs/angular-unittest
````
